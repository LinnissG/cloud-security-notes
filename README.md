## 1. The CIA Triad (Security Basics)

In cybersecurity, we often talk about the **CIA Triad**:

1. **Confidentiality** ‚Äì Only the right people can access the data.
2. **Integrity** ‚Äì The data is correct and has not been tampered with.
3. **Availability** ‚Äì The system and data are available when needed.

Example (Real Life):
- Bank app:
  - Confidentiality: Only you should see your balance.
  - Integrity: Your balance should be accurate.
  - Availability: The app should work when you need it.
